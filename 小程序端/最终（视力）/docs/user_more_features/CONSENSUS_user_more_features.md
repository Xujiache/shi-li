# CONSENSUS_user_more_features

## 1. 最终共识
- 个人中心 `更多` 区块新增 3 个协议入口：
  - `用户协议`
  - `隐私政策`
  - `儿童隐私政策`
- 保留现有 `关于与帮助` 与 `退出登录` 入口不变。
- 本次实现优先复用现有 `getTerms()` 接口与现有页面能力，不新增后端接口，不新建独立协议页面。

## 2. 验收标准
- `pages/user/index` 的“更多”区块可见 3 个新增协议入口。
- 点击不同入口后，可弹出对应协议内容。
- 协议内容为空时，弹窗可稳定展示 `暂无内容` 或默认占位文本。
- 原有 `关于与帮助`、`退出登录` 功能不受影响。
- 页面无新增报错，结构与视觉风格保持一致。

## 3. 技术共识
- 在 `pages/user/index/index.js` 内新增协议数据状态和弹窗展示方法。
- 通过 `getTerms()` 拉取后台协议配置，写入页面 `termsData`。
- 在 `pages/user/index/index.wxml` 中新增协议菜单项与协议弹窗结构。
- 在 `pages/user/index/index.wxss` 中补充协议弹窗样式，保持与当前页面风格一致。
