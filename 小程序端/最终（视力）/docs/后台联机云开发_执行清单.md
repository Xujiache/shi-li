# 后台联机云开发 — 执行清单

按顺序打勾执行，即可让 Vue 后台在本地和宝塔部署后都联接到微信云开发。

---

## 一、CloudBase 控制台（先做）

- [已完成] 登录 [腾讯云 CloudBase 控制台](https://tcb.cloud.tencent.com)，进入你的小程序云开发环境。
- [已完成] **身份认证** → 登录方式 → 开启 **匿名登录**（对接指南用匿名登录拿 access_token）。
- [已完成] **记下网关地址**（无需在云函数列表里点“开通 HTTP 触发”，没有这个按钮是正常的）：  
  - 在控制台首页或「概览」里找到当前环境的 **环境 ID**（形如 `cloud1-xxxxxx`）。  
  - 网关地址固定为：  
    `https://{环境ID}.api.tcloudbasegateway.com/v1/functions/admin_manager`  
    你当前环境 ID 为 `cloud1-1gwuiims1ab650b6`，则地址为：  
    `https://cloud1-1gwuiims1ab650b6.api.tcloudbasegateway.com/v1/functions/admin_manager`
- [ ] **安全来源 / 安全域名**（必配，否则浏览器会跨域）：  
  **操作路径**：打开 **[云开发 → 环境配置 → 安全来源](https://tcb.cloud.tencent.com/dev?#/env/safety-source)**（或 控制台左侧 **环境配置** → **安全来源**）。  
  1. 在「**安全域名**」区域点击 **【添加域名】**。  
  2. 在弹窗里填写域名（**不要带 `https://` 或 `http://`**），例如：  
     - 本地调试：填 `localhost`（系统默认已有）或 `localhost:5173`（带端口）。  
     - 部署后的后台：填纯域名，如 `admin.yourdomain.com`。  
  3. 点击 **【确定】**，约 1～2 分钟生效。  
  **说明**：支持完整域名、通配符（如 `*.yourdomain.com`）、带端口（如 `localhost:5173`）；每环境最多 50 个。

## 四、宝塔部署（联机到生产）

- [ ] 在 Vue 项目根目录配置 **`.env.production`**：  
  `VITE_CLOUDBASE_HTTP_URL` = 上面一、里记下的 **云函数网关地址**（同一串 URL）。
- [ ] 执行 `pnpm build`，将 **dist 内全部内容** 上传到宝塔站点根目录。
- [ ] 宝塔 Nginx 已按 [《宝塔面板部署步骤》](../宝塔面板部署步骤.md) 配置 `try_files $uri $uri/ /index.html;`。
- [ ] CloudBase 云函数 **CORS** 已添加你的后台访问域名（如 `https://admin.yourdomain.com`）。
- [ ] 浏览器访问该域名，用管理员账号登录，能进入后台即表示 **已联接到微信云开发**。
