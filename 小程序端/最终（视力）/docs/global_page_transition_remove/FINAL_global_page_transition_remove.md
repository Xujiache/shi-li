# FINAL_global_page_transition_remove

## 1. 交付摘要
- 已去除小程序端所有页面跳转时的全局弹性动画。
- 已同步简化主导航 `tabBar` 的切页反馈，避免在主入口切换时继续出现回弹感。

## 2. 主要改动
- `miniprogram/app.js`
  - 删除全局路由动作记录
  - 删除页面 `onShow` 中的转场关键帧执行
  - 保留页面访问埋点
- `miniprogram/app.wxss`
  - 删除页面级 fallback 动画
  - 删除容器子节点分层浮入动画
- `miniprogram/custom-tab-bar/*`
  - 删除 pulse 状态和关键帧动画
  - 删除抬升、缩放类反馈，仅保留轻量选中态

## 3. 结果收益
- 页面切换观感恢复为直接切换，不再有“弹一下”的多余感受。
- 主导航切换反馈更克制，不会和页面跳转叠加出明显动效。
- 改动集中在全局入口和公共组件层，后续维护成本低。
