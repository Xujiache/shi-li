# ALIGNMENT_questionnaire_fill_child_lock: 问卷填写页固定填写对象

## 1. 项目上下文
- 目标页面：`miniprogram/pages/questionnaire/fill/index`
- 当前状态：填写页存在“切换孩子”入口，填写对象卡片也更像可切换对象区域

## 2. 原始需求
- 该页面不允许更换孩子
- 填写对象应固定为上一页已确认的那个孩子
- 当前卡片仅展示实际填写对象，不作为切换入口

## 3. 需求理解
- 移除填写页中所有“切换孩子”入口
- 保留填写对象信息展示，但改成“固定档案展示”语义
- 无档案状态下不再提供“切换孩子”，仅保留完善档案引导

## 4. 决策
- 修改范围仅限 `fill/index.wxml`、`fill/index.js`
- 如 `head-btn` 样式不再被使用，可从样式中移除或保留但不影响功能
- 不改题目逻辑、不改提交逻辑
