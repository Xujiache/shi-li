# CONSENSUS_questionnaire_actionbar_float

## 1. 最终共识
- 问卷详情页和问卷填写页的关键操作按钮改为固定悬浮底部操作栏
- 按钮始终可见，无需滚动到底部再操作
- 页面内容区域需要额外增加底部留白，确保最后一段内容不会被悬浮栏覆盖

## 2. 验收标准
- `pages/questionnaire/detail/index`
  - “查看历史”“开始填写/继续填写”固定显示在页面底部
- `pages/questionnaire/fill/index`
  - “上一页”“下一页”“保存草稿”“提交问卷”固定显示在页面底部
- 两个页面滚动内容时，底部操作栏保持悬浮不跟随内容滚走
- 最后一个内容卡片不会被按钮遮挡
- 风格与现有问卷页面一致

## 3. 技术共识
- 通过固定定位和安全区间距实现
- 不新增 JS 交互逻辑，优先只改 WXML 和 WXSS
