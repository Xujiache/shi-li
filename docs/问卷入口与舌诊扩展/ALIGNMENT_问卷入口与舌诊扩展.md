# ALIGNMENT_问卷入口与舌诊扩展

## 1. 项目上下文分析
- 本次需求同时覆盖：
  - 小程序首页快捷入口
  - 小程序记录档案页
  - Node 后端检测记录模型
  - 管理后台检测记录新增/编辑
- 现有“舌形 / 舌色 / 舌苔”字段只存在于孩子档案模型 `children`，不在检测记录模型 `checkup_records`。

## 2. 原始需求
1. 首页增加问卷快捷跳转按钮
2. 记录档案新增舌形/舌色/舌苔填写
3. 管理后台支持新增/编辑这组舌诊字段

## 3. 用户关键确认
- 用户已确认：`舌形 / 舌色 / 舌苔` 要真正并入 `记录档案 / 检测记录` 模型，不回写孩子档案模型。

## 4. 可执行理解
- 首页快捷区新增“问卷中心”入口
- `checkup_records` 表新增：
  - `tongue_shape`
  - `tongue_color`
  - `tongue_coating`
- 小程序记录档案页新增三项选择并支持回填
- 管理后台检测记录弹窗新增三项选择并支持回填

## 5. 边界
- **包含**:
  - 新增/编辑/保存/回填
- **不包含**:
  - 数据看板展示舌诊字段
  - 历史记录列表额外展示舌诊字段
  - 问卷配置字段扩展
