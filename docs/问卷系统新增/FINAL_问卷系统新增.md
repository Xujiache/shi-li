# FINAL_问卷系统新增

## 一、交付结果

本次已为项目新增一套完整问卷体系，覆盖：

- 小程序端独立 `问卷` Tab
- 问卷中心、详情、填写、历史
- 后台问卷配置与答卷数据页
- 后端问卷配置、派发、草稿、提交、导出能力

## 二、核心新增点

1. 结构化年级字段正式进入业务主线。
2. 问卷可按年级、年级范围、班级、孩子等维度派发。
3. 问卷题型、规则、显隐逻辑、校验配置支持高级 JSON 承载。
4. 历史答卷保留快照，后台可查看完整填写内容。

## 三、当前样例

系统已初始化一份“`小学生心理健康评定量表（示例）`”问卷，便于直接联调问卷中心、填写和后台查看。
