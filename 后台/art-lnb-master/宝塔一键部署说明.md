## 二、宝塔部署三步

### 1. 添加网站
- 宝塔 → **网站** → **添加站点**
- 域名：填你的域名或服务器 IP
- 根目录：如 `/www/wwwroot/你的域名`

### 2. 上传并解压
- 方式 A：把 **`dist-宝塔部署.zip`** 上传到站点根目录，在宝塔文件管理中 **解压到当前目录**
- 方式 B：把 **`dist` 文件夹里的所有文件**（不是 dist 文件夹本身）上传到站点根目录

解压/上传后，站点根目录下应有：`index.html`、`assets` 文件夹、`favicon.ico`

### 3. 配置 Nginx（支持前端路由）
- 该站点 → **设置** → **配置文件**
- 在 `location /` 中确保有：`try_files $uri $uri/ /index.html;`
- 保存后 **重载配置**

示例：

```nginx
location / {
    try_files $uri $uri/ /index.html;
}
```

## 三、生产环境变量（重要）

本次构建使用的是项目中的 **`.env.production`** 配置。若你尚未配置或需要修改：

1. 编辑 `c:\Users\Administrator\Desktop\art-lnb-master\.env.production`
2. 确认 **`VITE_CLOUDBASE_HTTP_URL`** 为你的云函数 HTTP 网关地址（如 `https://cloud1-xxx.ap-shanghai.app.tcloudbase.com/v1/functions/admin_manager`）
3. 保存后重新执行构建：
   ```bash
   cd c:\Users\Administrator\Desktop\art-lnb-master
   pnpm run build
   ```
4. 再重新打包 zip 或重新上传 dist 内容
